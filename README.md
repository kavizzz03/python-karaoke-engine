# 🎵 Python Karaoke Lyrics Player

A real-time terminal-based karaoke system that plays MP3 songs and displays synchronized lyrics using AI transcription.

---

## 🚀 Features

- 🎧 Play MP3 songs (VLC backend)
- 🧠 AI transcription using OpenAI Whisper
- 🎤 Real-time lyric display in terminal
- ✨ Auto typing karaoke effect
- ⏱️ Micro-timed lyric breakdown (~0.5s chunks)
- 🇱🇰 Sinhala + English support
- 💻 Works on Python 3.14

🎯 How it works
Loads MP3 song
Uses Whisper AI to generate lyrics
Splits lyrics into small chunks
Displays synchronized karaoke-style text in terminal

⚠️ Limitations
Timing is approximate (not perfect Spotify-level sync)
Accuracy depends on audio quality
Terminal-based UI only
